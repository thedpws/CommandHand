INSTALL:
	1. USE MAKEFILE WITH MINGW COMPILER

USE:
	SET THE RGB VALUES.
	PERFORM A GESTURE AND PRESS A NUMBER KEY.
		THE NUMBER KEY CORRESPONDS TO THE ID AND FUNCTION OF THE GESTURE.
		GESTURE 1 IS LEFT-CLICK
		GESTURE 0 IS NEUTRAL
	PRESS Q ANYTIME TO QUIT


DEPENDENCIES:
	OPENCV
	QTGUI
